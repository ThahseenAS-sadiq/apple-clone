# 🍏 Apple Clone Website

A modern, responsive *Apple homepage clone* built using *React, **Vite, and **Tailwind CSS*.  
This project replicates the clean and minimal design of Apple’s official website while maintaining fast performance and reusable React components.

---

## 🚀 Tech Stack

- ⚛ *React* – Component-based UI library  
- ⚡ *Vite* – Lightning-fast frontend build tool  
- 🎨 *Tailwind CSS* – Utility-first CSS framework  
- 🔤 *React Icons* – For beautiful, lightweight icons
